Recurrent Neural Networks in Stock Price Prediction

A Python machine learning project which uses my own recurrent neural network to predict future stock prices. File IO and Argument Parsing with TCLAP

A C++ project which uses a third party C++ library, TCLAP, to process command line arguments. TCLAP is a framework for handling various kinds of arguments. Based on the arguments given, the program will read a plain text file, take the contents of the file and change them to either uppercase or lowercase, and then write the result to an output file. An exercise in the use of C++ file I/O and third-party libraries, as well as shell scripts. Please be patient as it takes a minute to run.
