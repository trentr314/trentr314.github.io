File IO and Argument Parsing with TCLAP

A C++ project which uses a third party C++ library, TCLAP, to process command line arguments. TCLAP is a framework for handling various kinds of arguments. Based on the arguments given, the program will read a plain text file and either uppercase or lowercase the contents of the file, then write the result to an output file. A simple exercise in the use of C++ file I/O and third-party libraries, as well as shell scripts.
